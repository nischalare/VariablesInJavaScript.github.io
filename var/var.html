var

Using var we can declare variables in JavaScript

When you use var, you can declare the same variable as many times as you like, but with let you can't.

var myName = 'Chris';

var myName = 'Bob';

var x = 5;

var y = 6;

var z = x + y;

If you want your code to run in older browser, you must use var.

The var keyword is used in all JavaScript code from 1995
